Title of the plot:- Comparison of Pollutants across years.
Type of the plot:- Parallel Coordinate Plot

Run the index.html file after passing the desired "State" 
in the plotParallelCoordinates function for the parallel coordinate plot
of pollutants across years and months of that state.

To see the comparison of pollutants of all the states at once, pass '' as the argument to the plotParallelCoordinates function.
